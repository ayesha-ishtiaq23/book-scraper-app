# 📚 Book Scraper & Analysis App

An interactive web scraping dashboard built with Python and Streamlit.

## 🔍 Features
- Scrapes live book data from books.toscrape.com
- Search books by title
- Filter books by price
- View top 5 most expensive books in a bar chart

## 🛠️ Tech Stack
- Python
- requests & BeautifulSoup (web scraping)
- pandas (data handling)
- Streamlit (interactive UI)
- matplotlib (charts)

## ▶️ How to Run
pip install requests beautifulsoup4 streamlit pandas matplotlib

streamlit run book_scraper.py
